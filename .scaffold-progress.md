### What was done

- Added `.gitignore`, `.env.example`, `docker-compose.yml` (MongoDB 7), `.golangci.yml`

### pkg/errors (sentinel errors + wrapping)
- **File:** `pkg/errors/errors.go`
- **Exported:** `ErrNotFound`, `ErrUnauthorized`, `ErrForbidden`, `ErrConflict`, `ErrAlreadyExists`, `ErrInvalidInput`, `ErrInternal`, `ErrUnavailable`
- **Functions:** `Wrap(err, msg)`, `Wrapf(err, fmt, args...)`, `Is()`, `As()`, `New()` -- all nil-safe
- **Pattern:** `fmt.Errorf("context: %w", err)` wrapping, checked via `errors.Is`/`errors.As`

### pkg/logging (Zap structured JSON)
- **File:** `pkg/logging/logging.go`
- **Exported:** `NewLogger(level string) *zap.Logger`, `NewDevelopmentLogger() *zap.Logger`
- **Context API:** `WithContext(ctx, logger) context.Context`, `FromContext(ctx) *zap.Logger`, `WithFields(ctx, ...zap.Field) context.Context`
- **Contract:** `FromContext` never returns nil (returns nop logger if none set)

### pkg/config (Viper config loading)
- **File:** `pkg/config/config.go`
- **Exported types:** `Config`, `ServerConfig`, `MongoDBConfig`, `LoggingConfig`, `ShutdownConfig`
- **Functions:** `Load(configPath string) *Config`, `LoadFromEnv() *Config`
- **Env prefix:** `ZEE6DO_` (e.g. `ZEE6DO_MONGODB_URI`, `ZEE6DO_SERVER_GRPC_PORT`)
- **Validation:** Panics on missing `mongodb.uri`, invalid port (1-65535), invalid environment
- **Defaults:** port=50051, env=development, db=zee6do, pool=100, connect_timeout=10s, timeout=30s, server_selection_timeout=5s, log_level=info, grace_period=30s, drain_interval=10s

## Milestone 2: Database + Server Lifecycle
**Status:** Complete

### internal/database (MongoDB connection + migrations)
- **Files:** `internal/database/mongodb.go`, `internal/database/migration.go`, `internal/database/wire.go`
- **MongoClient exported API:** `NewMongoClient(ctx, *config.Config, *zap.Logger) (*MongoClient, error)`, `Database() *mongo.Database`, `Client() *mongo.Client`, `HealthCheck(ctx) error`, `Close(ctx) error`
- **Pool config:** MaxPoolSize, ConnectTimeout, Timeout (v2 client-level), ServerSelectionTimeout, RetryWrites, RetryReads
- **MigrationRunner API:** `NewMigrationRunner(db, logger)`, `Register(Migration)`, `Run(ctx) error`
- **Migration struct:** `{ID string, Description string, Up MigrationFunc}` where `MigrationFunc = func(ctx, *mongo.Database) error`
- **Tracking:** `_migrations` collection, records `{_id, description, applied_at}`, sorted by ID, skips already-applied
- **Wire:** `database.ProviderSet` provides `NewMongoClient`

### internal/server (health + shutdown)
- **Files:** `internal/server/health.go`, `internal/server/shutdown.go`, `internal/server/wire.go`
- **HealthChecker API:** `NewHealthChecker() *HealthChecker`, `IsHealthy() bool`, `SetUnhealthy()`
- **Shutdownable interface:** `Close(ctx context.Context) error` -- implement this to register for shutdown
- **ShutdownManager API:** `NewShutdownManager(logger, health, drainInterval, gracePeriod)`, `Register(name, Shutdownable)`, `Shutdown()`
- **Shutdown sequence:** unhealthy -> drain wait -> close components in order -> done (30s total timeout)
- **Wire:** `server.ProviderSet` provides `NewHealthChecker`

### migrations
- **File:** `migrations/migrations.go`
- **Function:** `Register(runner *database.MigrationRunner)` -- registers all migrations
- **Initial migration:** `001_create_initial_indexes` -- users unique index on user_id, tasks compound index on (user_id, status, priority)

### cmd/server (entry point + Wire DI)
- **Files:** `cmd/server/main.go`, `cmd/server/app.go`, `cmd/server/wire.go`, `cmd/server/wire_gen.go`
- **App struct:** holds Config, Logger, MongoClient, HealthChecker, ShutdownManager
- **Wire injector:** `InitializeApp(ctx, *config.Config) (*App, error)` -- wires logger, mongo, health, app
- **main.go flow:** parse flags -> load config -> InitializeApp -> run migrations -> wait for SIGINT/SIGTERM -> graceful shutdown
- **Logger selection:** development env -> dev logger, otherwise production JSON logger

### Config / DevOps

- **docker-compose.yml:** MongoDB 7 on port 27017, healthcheck, named volume
